# Calculadora de Notas

Projeto simples em Python utilizando testes automatizados com pytest.

## Funcionalidades

- cálculo de média ponderada;
- validação de notas;
- verificação da situação do aluno.

## Regra da média

A terceira nota possui peso 2.

Fórmula:

(N1 + N2 + 2 × N3) / 4
